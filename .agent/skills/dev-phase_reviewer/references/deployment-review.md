# 部署阶段 Review 清单

适用阶段: `deployment`

## 配置完整性

- [ ] **环境变量** — 所有必需的环境变量已定义（列出清单）
- [ ] **数据库连接** — 生产环境数据库连接配置正确
- [ ] **第三方服务** — API key、webhook URL 等已配置
- [ ] **域名/URL** — 生产环境 URL 已配置
- [ ] **部署配置文件** — `render.yaml` / `docker-compose.yml` / CI 配置存在且完整

## 安全合规

- [ ] **无明文密钥** — 所有密钥通过环境变量或 Secret Manager 管理
- [ ] **HTTPS** — 启用 TLS/SSL
- [ ] **CORS** — 正确配置允许的来源（非 `*`）
- [ ] **Rate Limiting** — API 端点有速率限制
- [ ] **安全头** — Content-Security-Policy, X-Frame-Options 等已配置
- [ ] **.env 不在版本控制中** — `.gitignore` 包含 `.env`
- [ ] **依赖安全** — `uv run pip-audit` / `npm audit` 无 HIGH/CRITICAL 漏洞

## 可靠性

- [ ] **健康检查** — `/health` 端点存在且返回正确状态
- [ ] **回滚方案** — 有明确的回滚步骤（文档或脚本）
- [ ] **数据库迁移** — 迁移脚本可正向/反向执行
- [ ] **错误处理** — 全局异常处理器已配置，不泄露内部信息
- [ ] **超时配置** — 请求超时、数据库连接超时已设置

## 可观测性

- [ ] **日志** — 结构化日志已配置（JSON 格式）
- [ ] **日志级别** — 生产环境使用 INFO 级别（非 DEBUG）
- [ ] **监控** — 关键指标有监控（CPU、内存、请求延迟、错误率）
- [ ] **告警** — 关键故障有告警通知

## 性能

- [ ] **构建优化** — 生产构建已启用压缩和树摇
- [ ] **静态资源** — CDN 或缓存策略已配置
- [ ] **数据库** — 生产环境连接池大小合理

## 文档

- [ ] **部署文档** — README 或 `docs/deployment.md` 包含部署步骤
- [ ] **环境变量文档** — 所有环境变量有说明（`.env.example`）
- [ ] **运维手册** — 常见问题排查步骤已记录

## 审批标准

- ✅ **批准**: 安全合规 + 配置完整 + 健康检查通过
- ⚠️ **条件通过**: 非生产环境部署，可接受部分监控缺失
- ❌ **阻止**: 安全问题、无健康检查、无回滚方案

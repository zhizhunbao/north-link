# North Link 跨境货源通 — 需求分析文档

> 版本: V1.0 | 更新时间: 2026-02-28 | 作者: Alice (Product Manager)

## 1. 项目概述

### 1.1 背景

加拿大货源方在跨境贸易中面临 **价格信息分散、利润计算复杂、物流关税不透明、下游客户难找、全流程无法标准化管理** 等痛点。现有工作流高度依赖人工Excel表格和微信沟通，效率低、易出错。

### 1.2 产品定位

面向加拿大货源方的 **全自动化跨境套利、比价、利润核算、物流管理、国内商户对接** 一站式 SaaS 管理系统。

### 1.3 目标

- 将货源查找、利润计算、商户匹配、物流下单的全流程从 **手动操作** 转为 **系统自动化**
- 实现 1 人管理 50+ 单日订单
- 零技术门槛，开箱即用

### 1.4 范围

- **在范围内**: 比价、利润计算、商户管理、物流对接、订单管理、智能推荐
- **不在范围内**: 支付结算、仓储囤货、走私/违禁品、国家管制类商品

## 2. 目标用户

### 2.1 用户画像

| 用户类型                    | 描述                                                 | 核心需求                             |
| --------------------------- | ---------------------------------------------------- | ------------------------------------ |
| **核心用户** — 加拿大货源商 | 个人或小型团队，在加拿大本地采购低价商品卖到中国     | 找到高利润商品、一键算利润、快速出货 |
| **下游用户** — 国内商家     | 数码商家、改卡工厂、AI算力公司、代购、实体店、批发商 | 稳定的加拿大货源、合理报价、快速到货 |
| **合作方** — 加拿大货代     | 货运代理、清关公司、物流服务商                       | 获取更多客户订单、简化对接流程       |

### 2.2 用户场景

| #   | 场景         | 用户行为                   | 系统响应                                       |
| --- | ------------ | -------------------------- | ---------------------------------------------- |
| S1  | 每日找货     | 打开系统查看今日推荐       | 展示 TOP5 高利润商品 + 推荐出货商户 + 推荐物流 |
| S2  | 发现低价商品 | 手动录入发现的低价商品信息 | 自动匹配国内售价，计算利润，推荐商户           |
| S3  | 确认采购     | 选择商品下单采购           | 记录采购订单，关联商品和成本                   |
| S4  | 安排发货     | 选择商品 + 收货商户        | 自动推荐物流方案，一键下单货代                 |
| S5  | 跟踪物流     | 查看物流状态               | 实时展示：加拿大→清关→国内→签收                |
| S6  | 查看报表     | 查看本周/本月利润          | 展示营收、利润、成本占比、单品排行             |

## 3. 约束条件

### 业务约束

- 不涉及走私、违规清关、违禁品
- 不做资金池、不参与支付结算（记录回款状态 ≠ 参与支付）
- 不囤货、不做仓储
- 不销售国家管制类商品

### 技术约束

- 后端: Python 3.12+ / FastAPI / SQLAlchemy 2.0
- 前端: React 19+ / TypeScript / Vite / Ant Design 5
- 数据库: PostgreSQL 16 + Redis 7
- 部署: Docker + Render.com (MVP)
- 包管理: uv (Python) / npm (Node)

## 4. 非功能需求（方向性）

| 类别     | 目标                             |
| -------- | -------------------------------- |
| 易用性   | 全中文界面，零技术门槛           |
| 性能     | 页面加载 < 2s，利润计算 < 3s     |
| 数据更新 | MVP: 手动导入；V1.5: 每日 4-6 次 |
| 安全性   | 敏感数据加密，JWT 认证           |
| 兼容性   | 桌面 + 手机浏览器（响应式）      |

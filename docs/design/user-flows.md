# North Link 跨境货源通 — 用户流程

> 版本: V1.0 | 更新时间: 2026-02-28 | 作者: Bella (UI/UX Designer)

## 1. 核心用户流程

### Flow 1: 每日找货 (S1 — MVP 核心)

```mermaid
flowchart TD
    A[打开系统] --> B[仪表盘: 今日 TOP5 推荐]
    B --> C{感兴趣?}
    C -->|是| D[点击商品 → 商品详情]
    C -->|否| E[进入比价中心手动搜索]
    D --> F[查看多平台价格 + 利润计算]
    F --> G{利润满意?}
    G -->|是| H[查看推荐商户 + 推荐物流]
    G -->|否| I[返回列表继续浏览]
    H --> J[收藏 / 直接发货]
    E --> K[筛选分类 + 排序]
    K --> D
```

### Flow 2: 手动录入发现的低价商品 (S2)

```mermaid
flowchart TD
    A["发现低价商品"] --> B["点击 手动录入"]
    B --> C["填写: 商品名/价格/渠道/备注"]
    C --> D["系统自动匹配国内售价"]
    D --> E["展示利润计算结果"]
    E --> F{"利润可观?"}
    F -->|是| G["推荐最高报价商户"]
    F -->|否| H["标记低利润, 留观"]
    G --> I["联系商户或收藏"]
```

### Flow 3: 安排发货 (S4)

```mermaid
flowchart TD
    A["选择要发的商品"] --> B["选择收货商户"]
    B --> C["系统推荐物流方案"]
    C --> D["展示: 最便宜/最快/包税/批量优"]
    D --> E{"选择方案"}
    E --> F["确认商品信息 + 运费税费"]
    F --> G["一键下单, 生成取件码"]
    G --> H["物流跟踪页面"]
```

### Flow 4: 物流跟踪 (S5)

```mermaid
flowchart TD
    A["物流管理页面"] --> B["在途订单列表"]
    B --> C["点击订单查看详情"]
    C --> D["物流轨迹时间线"]
    D --> E["加拿大取件 完成"]
    E --> F["海关清关中"]
    F --> G["国内派送"]
    G --> H["签收确认"]
```

### Flow 5: 商户管理

```mermaid
flowchart TD
    A["商户管理页面"] --> B["商户列表 - 按品类分组"]
    B --> C{"操作"}
    C -->|新增| D["填写商户信息表单"]
    C -->|查看| E["商户详情: 报价+历史"]
    C -->|编辑| F["修改商户信息"]
    D --> G["保存, 列表刷新"]
    E --> H{"要更新报价?"}
    H -->|是| I["录入新报价"]
    H -->|否| J["返回列表"]
```

### Flow 6: 查看报表 (S6)

```mermaid
flowchart TD
    A["进入报表中心"] --> B["选择时间范围: 今日/本周/本月"]
    B --> C["展示营收/利润/成本占比"]
    C --> D["查看单品排行榜"]
    D --> E{"需要导出?"}
    E -->|是| F["选择格式: Excel/PDF"]
    F --> G["下载报表文件"]
    E -->|否| H["返回仪表盘"]
```

## 2. 页面跳转关系

```mermaid
flowchart LR
    Dashboard["仪表盘"] --> PriceCenter["比价中心"]
    Dashboard --> Merchants["商户管理"]
    Dashboard --> Logistics["物流管理"]
    Dashboard --> Reports["报表中心"]

    PriceCenter --> ProductDetail["商品详情"]
    ProductDetail --> Merchants
    ProductDetail --> Logistics

    Merchants --> MerchantDetail["商户详情"]

    Logistics --> ShipOrder["发货下单"]
    Logistics --> Tracking["物流跟踪"]
```

## 3. 错误与边界场景

| 场景             | 处理方式                        |
| ---------------- | ------------------------------- |
| 无推荐数据       | 空状态: 引导用户手动录入商品    |
| 利润计算参数缺失 | 使用默认参数 + 提示用户去设置页 |
| 国内价格未匹配到 | 提示 "暂无匹配"，允许手动输入   |
| 网络中断         | 离线缓存最近数据 + 重试提示     |
| 物流状态更新延迟 | 展示最后更新时间 + 刷新按钮     |

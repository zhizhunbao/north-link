## 2. 导航结构

### 2.1 桌面端 — 左侧边栏导航

```
┌─ Logo ──────────────────────────────────────────┐
│                                                  │
│  📊 仪表盘                                       │
│  🔍 比价中心                                     │
│  👥 商户管理                                     │
│  🚚 物流管理                                     │
│  📦 订单管理                                     │
│  📈 报表中心                                     │
│  ─────────────                                   │
│  ⚙️ 设置                                         │
│                                                  │
│  [用户头像]  用户名                               │
└──────────────────────────────────────────────────┘
```

### 2.2 移动端 — 底部标签栏

```
┌─────────────────────────────────────┐
│           页面内容区域               │
│                                     │
│                                     │
├─────┬─────┬─────┬─────┬─────────────┤
│ 首页 │ 比价 │ 商户 │ 物流 │   更多   │
│  📊  │  🔍  │  👥  │  🚚  │   ···   │
└─────┴─────┴─────┴─────┴─────────────┘
```

"更多" 展开：订单、报表、设置

### 2.3 面包屑规则

- 一级页面不显示面包屑
- 二级及以下显示：`首页 / 比价中心 / RTX 4090 详情`
- 移动端用返回箭头替代面包屑

## 3. 内容层级

### 3.1 仪表盘

| 优先级    | 内容                  | 位置     |
| --------- | --------------------- | -------- |
| Primary   | 今日 TOP5 推荐        | 页面顶部 |
| Secondary | 统计卡片（利润/订单） | 推荐下方 |
| Tertiary  | 最近操作记录          | 页面底部 |

### 3.2 比价中心

| 优先级    | 内容                  | 位置     |
| --------- | --------------------- | -------- |
| Primary   | 商品列表（价格+利润） | 页面主体 |
| Secondary | 筛选/排序工具栏       | 列表上方 |
| Tertiary  | 分页/加载更多         | 列表底部 |

### 3.3 商品详情页

| 优先级    | 内容                    | 位置     |
| --------- | ----------------------- | -------- |
| Primary   | 利润计算结果 + 风险等级 | 页面顶部 |
| Secondary | 多平台价格对比表        | 中部     |
| Tertiary  | 推荐商户 + 推荐物流     | 底部     |
